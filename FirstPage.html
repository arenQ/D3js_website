Introduction

D3.js is a JavaScript library for manipulating documents based on data. D3 helps you bring data to life using HTML, SVG, and CSS. D3’s emphasis on web standards gives you the full capabilities of modern browsers without tying yourself to a proprietary framework, combining powerful visualization components and a data-driven approach to DOM manipulation.I made this website for calling more people to learn and use it.

Source

demo和代码

D3js官网

https://d3js.org/

在这里能获得d3.js的最新版本，同时还拥有大量demo和代码示例

官方API文档

https://github.com/d3/d3/blob/master/API.md

通过它可以了解到D3API的使用

D3开发者Mike Bostock的博客

https://bl.ocks.org/mbostock

包含大量在线示例，介绍了很多D3的编程技巧，讲解了部分API的使用，还探讨了许多使用D3会遇到的问题的解决方案

D3开发者Jason Davies的个人网站

https://www.jasondavies.com/

包含很多在线示例，涉及一些API的形象解释。扩展了D3的地理投影。讲解了很多D3的实现原理如贝塞尔曲线的动态演示等。还有一些像词云这样的使用插件

2000多个在线案例汇总

http://christopheviau.com/d3list/gallery.html

作者收集的这些可视化案例大都可以在线运行，并且已经按照类别组织好，可以通过关键词来筛选案例。

SVG在线教程

http://tutorials.jenkov.com/svg/index.html

讲解SVG的基本功能，案例都可以在线演示。

书籍

‹‹数据可视化实战（使用D3设计交互式图表）››

Scott Murry著，李松峰译。这本书写得浅显易懂，是入门D3的经典书籍。其中也有涉及一些API的简要介绍

‹‹D3.js In Action››

Elijah Meeks著。设计D3数据可视化的方方面面，特别是数据可视化和D3.js的原理，定制组件，大数据可视化等内容很经典。内容详实丰富

‹‹Mastering D3.js››

Pablo Navarro著。其中讲解了比较实用的地理可视化和实时数据可视化，包含部分API的介绍
